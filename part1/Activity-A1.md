# A1. Discover security concepts used on campus.

## Objective
The objective of this activity is to identify and analyze cybersecurity concepts implemented within the university campus environment.

---

## Methodology
To complete this activity, I conducted observations around campus and reviewed available university services and systems. This included:
- Observing physical access control systems in buildings
- Identifying surveillance systems such as CCTV
- Exploring the SafeZone mobile application
- Reviewing publicly available university safety information

---

## Findings

### 1. Physical Access Control

Physical access control is implemented across campus buildings using ID card systems.

- **Identity (Who you are):**
  - Student
  - Staff
  - Visitor

- **Authentication (What you have):**
  - Campus ID card used to scan at entry points

- **Authorization (What you can access):**
  - Access is restricted based on role and time (e.g., certain labs only accessible to specific students or staff)

#### Potential Vulnerabilities
- Tailgating (unauthorized individuals following authorized users)
- ID cards cannot verify the actual identity of the holder without manual checks
- Lost or stolen cards may be misused

---

### 2. CCTV Surveillance System

CCTV cameras are deployed across campus to monitor activities.

- Typically placed in:
  - High positions
  - Wide-angle coverage areas
  - High-traffic zones (e.g., entrances, transport hubs)

- Operates continuously (24/7 monitoring)

#### Potential Vulnerabilities
- Presence of blind spots
- Limited ability to prevent incidents (mainly reactive)
- Privacy concerns if misused

---

### 3. SafeZone Mobile Application

The SafeZone app is used to enhance campus safety for students, staff, and visitors.

#### Key Features
- Emergency assistance with real-time location tracking
- First aid request functionality
- Non-urgent help and security escort
- Wellbeing support links
- Check-in timer for lone workers/students
- Safety notifications
- Campus navigation

#### Security Concepts
- Location tracking (context-aware security)
- User identification through app login
- Real-time communication with security services

#### Potential Vulnerabilities
- Dependence on mobile device availability
- Privacy risks related to location tracking
- Possible misuse if unauthorized access to user account occurs


---

## Analysis

This activity demonstrates how multiple layers of security are applied on campus, combining physical security and digital systems.

- Physical access control enforces **authentication and authorization**
- CCTV provides **monitoring and deterrence**
- SafeZone integrates **digital security with real-time response systems**

However, each system has limitations. For example, access cards rely on possession rather than identity verification, and CCTV cannot prevent incidents in real time. This highlights the importance of **layered security (defense in depth)**.
